# TN Thread Order 1.0.0

Initial public release.

## Download

Download the attached file:

- `tn-thread-order-1.0.0.xpi`

## SHA256

`e1d8dc78f70ab936a05de0b5ef63fdd00cb7de8d40d8d9826120928dd8bfa442`

## Compatibility

- Thunderbird 152.*

## Highlights

- Newest-first threaded conversations.
- Newest message shown as the main row in collapsed threads.
- Expanded threads ordered newest-first.
- Preview pane synchronized with the projected order.
- Card View relationship indicators.
- Table View support using Thunderbird-native layout, icons, indentation, states, and account colors.
- Unified Inbox and cross-account relationship handling.
- English and Portuguese (Brazil) localization.

## Installation

1. Download the XPI file attached to this release.
2. In Thunderbird, open **Add-ons and Themes**.
3. Click the gear icon.
4. Choose **Install Add-on From File…**.
5. Select the downloaded XPI file.

## Notes

This extension uses Thunderbird Experiment APIs because Thunderbird does not currently expose the required message-list internals through standard WebExtension APIs.
