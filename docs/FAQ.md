# FAQ

## What does TN Thread Order do?

It changes Thunderbird threaded conversations to show the newest message first.

## Which Thunderbird version is supported?

Version 1.0.0 targets Thunderbird 152.*.

## Does it support Card View and Table View?

Yes. Card View includes relationship indicators. Table View keeps Thunderbird-native layout and icons.

## Does it support Unified Inbox?

Yes. It includes handling for Unified Inbox and cross-account threads.

## Why is it distributed on GitHub?

The extension depends on Thunderbird Experiment APIs because the required message-list internals are not available through standard WebExtension APIs.

## Where should I report problems?

Use GitHub Issues for reproducible bugs and Discussions for questions, compatibility reports, and suggestions.
